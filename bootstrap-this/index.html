What is Bootstrap

Bootstrap is a framework that helps you to create responsive websites. Responsive means that when the size of the screen changes, example web 1280x800px and IPhone 12 Pro mobile 428x926px. With the html being responsive the content will look nice at both sizes of the screen. Bootstrap has to be incorporate into your html file, this can be done one of two ways. First it can be imported into your code by downloading and linking it locally to your code. Second way is to link it from the Bootstrap website on your browser, copy the code from the website and add it within your head tags.

   

Colors

   

Bootstrap colors can be used in a few different ways, background, text, links, and componenets. Each serving a different purpose to your page. Text colorize the text on the page using "text" followed by a dash and the color's name. An example of this is text-primary for blue, text-secondary is grey, text-success is green, etc. when used as a class it will change the color of the text within that class.
