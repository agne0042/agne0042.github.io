What is Bootstrap

Bootstrap is a framework that helps you to create responsive websites. Responsive means that when the size of the screen changes, example web 1280x800px and IPhone 12 Pro mobile 428x926px. With the html being responsive the content will look nice with any amount of the screens height and width being changed. Bootstrap has to be incorporate into your html file, this can be done one of two ways. First it can be imported into your code by downloading and linking it locally to your code. Second way is to link it from the Bootstrap website on your browser, copy the code from the website and add it within your head tags.

   

Colors

   

Bootstrap colors can be used in a few different ways, background, text, links, and componenets. Each serving a different purpose to your page. Text colorize the text on the page using "text" followed by a dash and the color's name. An example of this is text-primary for blue, text-secondary is grey, text-success is green, etc. when used as a class it will change the color of the text within that class.

   

Navigation Bar

   

Bootstrap offers easy to use navigation bars (or nav bar) with plenty of ways to customize them. To create a standard nav bar use the tag nav class="nav bar nav-bar default". Or customize the nav bar to how you want it to look. With Bootstrap you can quickly add a nav bar that has a drop down element by adding class="dropdown-toggle" data-toggle="dropdown". If you want to add a Sign Up and Login link to your nav bar simply add a href="#" span class="glyphicon glychicon-user" after closing span add some text "Sign Up". Repeat but instead use class"glyphicon glychicon-log-in" with some text "Login"

   

Cards

   

A card in Bootstra is a boardered box. Different cards can include options for a image, header, footer, content, colors, etc. A basic card card be created by giving a div class="card" with will create a boardered box. However there is not much to look at without adding more elements. To add more content you create a div class="card" amd add child elements to the parnent such as div class="header", div class="content", or div class="footer", then filling in what you want to content to say for your header, content, and footer.
